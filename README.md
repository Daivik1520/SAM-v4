## ğŸ› **Troubleshooting**

<details>
<summary><b>âŒ Common Issues & Solutions</b></summary>

### Voice Recognition Issues
```bash
# Problem: Microphone not detected
# Solution: Check Windows Sound settings
1. Right-click sound icon in system tray
2. Select "Sounds" â†’ "Recording" tab
3. Enable and set microphone as default
4. Test microphone levels
```

### Permission Issues
```bash
# Problem: Power commands don't work
# Solution: Run as Administrator
1. Right-click Command Prompt/PowerShell
2. Select "Run as Administrator"
3. Navigate to SAM-v3 directory
4. Run: python SAM.py
```

### Display Control Issues
```bash
# Problem: Brightness control not working
# Solutions:
1. Update display drivers
2. Check if external monitor supports DDC/CI
3. Use Windows display settings as fallback
4. Try different brightness commands
```

### Audio Device Issues
```bash
# Problem: Wrong audio device controlled
# Solution: Set default device
1. Right-click sound icon â†’ "Open Sound settings"
2. Choose correct output device
3. Restart SAM v3
```

### Missing Files Warnings
```bash
# Problem: user_commands.json or hotwords.json warnings
# Solution: These are optional files
1. SAM works with defaults - no action needed
2. Create custom files if you want personalization
3. Use examples in Configuration section
```

### macOS-Specific Issues
```bash
# Problem: Brightness control not working
# Solution: Install brightness CLI
brew install brightness

# Problem: Permission denied for system controls
# Solution: Grant Terminal/Python accessibility access
1. Open System Settings â†’ Privacy & Security â†’ Accessibility
2. Add Terminal.app and/or Python to the allowed list

# Problem: TTS voice sounds robotic
# Solution: Download enhanced voices
1. Open System Settings â†’ Accessibility â†’ Spoken Content
2. Click "System Voice" â†’ Manage Voices
3. Download "Samantha" or "Alex" for better quality

# Problem: Audio control not working
# Solution: SAM uses osascript which requires no special permissions
# If issues persist, check sound preferences in System Settings
```
